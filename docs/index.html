TL;DR

We study action-conditioned selective latent updates for robot manipulation. Instead of dense full-state updates, we update only a small subset of local latent states likely to be affected by the current action.

World Model Selective Update Robot Manipulation

Problem

Dense latent updates can spend substantial computation on regions not causally impacted by the action. We ask whether action-conditioned local updates can preserve control quality while improving update efficiency.

  1. Can we identify action-relevant local latent states?
  2. Can we update only those states without hurting task success?
  3. When do explicit updates beat selection-only baselines?

Method Overview

z_t --(policy)--> a_t
z_t, a_t --(selector)--> M_t
z_t, a_t, M_t --(transition)--> z_{t+1} (selective update)
  1. Encode current observation into latent tokens.
  2. Predict an action-conditioned mask over local tokens.
  3. Apply transition deltas only on selected tokens.
  4. Use the updated state for control.
